📘 AetherLang v1.2 — Complete Guide & Reference
───────────────────────────────────────────────
Built on Python. Replaced Python. Fully Yours.

══════════════════════════════
🧱 BASIC STRUCTURE
══════════════════════════════
• Statements run top to bottom.
• Blocks (like repeat/if) must be followed by ":" and separated by empty lines.
• Strings must use "quotes".

══════════════════════════════
📤 OUTPUT & INPUT
══════════════════════════════
• text("Hello")        → Output
• input("Your name?")  → Stores in _input
• int("123")           → Integer
• boo("true")          → Boolean

══════════════════════════════
🪄 VARIABLES
══════════════════════════════
• name = "Adam"
• age = int("25")
• x = random(1, 5)
• y = randomDecimal(0.5, 1.5)
• math = solve("3+5")

══════════════════════════════
🔁 LOOPS & CONDITIONS
══════════════════════════════
repeat 3:
    text("Hi")

if _input == "yes":
    text("Welcome")
other:
    text("Goodbye")
otherwise:
    text("Try again")

══════════════════════════════
🧮 MATH
══════════════════════════════
• solve("3 + 5")      → 8
• sqrt(16), round(4.5)
• pi, e, sin(pi), abs(-1)

══════════════════════════════
💾 STORAGE
══════════════════════════════
• store("username", "Adam")
• recall("username")       → Adam

- Stored in 'aether_storage.json'
- Survives restarts

══════════════════════════════
🛑 LIMITATIONS
══════════════════════════════
• No nested if/loops (yet)
• No function definitions
• All variables are global
• Quotes required for strings
